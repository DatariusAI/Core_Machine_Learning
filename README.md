## Overview

This repository serves as a comprehensive portfolio of core machine learning concepts and implementations. Each notebook walks through theory, implementation, and evaluation of fundamental ML algorithms -- covering supervised learning, unsupervised learning, and everything in between.

The goal is to build a strong foundation in classical machine learning before advancing into deep learning and specialized domains.

## Getting Started

### Prerequisites

```bash
pip install numpy pandas scikit-learn matplotlib seaborn jupyter
```

### Running Notebooks

**Locally:**
```bash
jupyter notebook regression_colab_file.ipynb
```

**On Google Colab:**

Click the "Open in Colab" badge at the top of each notebook, or upload the `.ipynb` file directly to [Google Colab](https://colab.research.google.com).
